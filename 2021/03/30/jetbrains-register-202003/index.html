jetbrains系列注册2020.03

发表于 2021-03-30 分类于 破解 阅读次数： Valine：
本文字数： 4.3k 阅读时长 ≈ 4 分钟

下载

  • 直接jetbrains官网下载（webStorm、phpStorm、idea、pyCharm）

  • 安装完成后，选择 ”Evaluate for free“ ，点击 “Evaluate” 进入 IDE 中，新建或打开一个项目。

安装插件

打开插件中心 File -> Settings -> Plugins，设置插件中心仓库，如下图所示

设置仓库地址：https://repo.idechajian.com ，如下图所示

点击OK，便可以在插件中心，搜索关键字BetterIntellij，如下图所示

直接安装插件，安装成功后会提示你，这时需要重启IDE，必须要重启(不行多重启几次)

注册

重启成功后Help -> Register，选择添加激活KEY，如下图所示

激活key

激活成功
